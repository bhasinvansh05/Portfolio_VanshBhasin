## 💻 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bhasinvansh05/Portfolio_VanshBhasin.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
