# Portfolio_VanshBhasin

A personal portfolio website built with React and Vite.

## Live Website

GitHub Pages URL:
`https://bhasinvansh05.github.io/Portfolio_VanshBhasin/`

## Tech Stack

- React 18
- Vite 6
- TypeScript
- Tailwind CSS 4
- Radix UI components
- Framer Motion (`motion`)
- Material UI (MUI)

## Project Structure

```text
src/
	main.tsx
	app/
		App.tsx
		components/
			Hero.tsx
			About.tsx
			Research.tsx
			Projects.tsx
			Skills.tsx
			Experience.tsx
			Achievements.tsx
			Resume.tsx
			Contact.tsx
			Navigation.tsx
			Footer.tsx
			CustomCursor.tsx
			InteractiveBackground.tsx
			ui/
	styles/
		index.css
		fonts.css
		tailwind.css
		theme.css
```

The app entry point is `src/main.tsx`, which renders `src/app/App.tsx`.

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## Available Scripts

- `npm run dev` - starts Vite dev server
- `npm run build` - builds the production bundle
- `npm run preview` - previews the production bundle locally
